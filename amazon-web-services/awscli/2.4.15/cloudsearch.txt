NAME
       cloudsearch -

DESCRIPTION
       You use the Amazon CloudSearch configuration service to create, config-
       ure, and manage search domains. Configuration service requests are sub-
       mitted  using  the  AWS  Query protocol. AWS Query requests are HTTP or
       HTTPS requests submitted via HTTP GET or POST with  a  query  parameter
       named Action.

       The  endpoint  for  configuration  service requests is region-specific:
       cloudsearch.*region*    .amazonaws.com.     For     example,     cloud-
       search.us-east-1.amazonaws.com. For a current list of supported regions
       and endpoints, see Regions and Endpoints .
