NAME
       update-entity -

DESCRIPTION

       See also: AWS API Documentation

       See 'aws help' for descriptions of global parameters.

SYNOPSIS

          [--component-updates <value>]
          [--description <value>]
          --entity-id <value>
          [--entity-name <value>]
          [--parent-entity-update <value>]
          --workspace-id <value>
          [--cli-input-json | --cli-input-yaml]
          [--generate-cli-skeleton <value>]
