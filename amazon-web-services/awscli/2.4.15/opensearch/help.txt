NAME
       opensearch -

DESCRIPTION
       Use  the  Amazon OpenSearch configuration API to create, configure, and
       manage Amazon OpenSearch Service domains.

       For sample code  that  uses  the  configuration  API,  see  the  Amazon
       OpenSearch  Service  Developer  Guide  . The guide also contains sample
       code for sending signed HTTP requests to the OpenSearch APIs .

       The endpoint for configuration  service  requests  is  region-specific:
       es.*region*  .amazonaws.com.  For  example, es.us-east-1.amazonaws.com.
       For a current list of supported regions and endpoints, see Regions  and
       Endpoints .
