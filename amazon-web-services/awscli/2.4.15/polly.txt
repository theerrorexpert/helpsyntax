NAME
       polly -

DESCRIPTION
       Amazon  Polly  is a web service that makes it easy to synthesize speech
       from text.

       The Amazon Polly  service  provides  API  operations  for  synthesizing
       high-quality  speech  from  plain text and Speech Synthesis Markup Lan-
       guage (SSML), along with managing pronunciations lexicons  that  enable
       you to get the best results for your application domain.

AVAILABLE COMMANDS
       o delete-lexicon

       o describe-voices

       o get-lexicon

       o get-speech-synthesis-task

       o help

       o list-lexicons

       o list-speech-synthesis-tasks

       o put-lexicon

       o start-speech-synthesis-task

       o synthesize-speech
