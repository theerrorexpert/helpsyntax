NAME
       appstream -

DESCRIPTION
       This  is  the  Amazon  AppStream 2.0 API Reference . This documentation
       provides descriptions and syntax for each of the actions and data types
       in  AppStream 2.0. AppStream 2.0 is a fully managed, secure application
       streaming service that lets you stream desktop  applications  to  users
       without rewriting applications. AppStream 2.0 manages the AWS resources
       that are required to host and run your applications,  scales  automati-
       cally, and provides access to your users on demand.

       NOTE:
          You  can call the AppStream 2.0 API operations by using an interface
          VPC endpoint (interface endpoint). For more information, see  Access
          AppStream  2.0  API Operations and CLI Commands Through an Interface
          VPC Endpoint in the Amazon AppStream 2.0 Administration Guide .

       To learn more about AppStream 2.0, see the following resources:

       o Amazon AppStream 2.0 product page

       o Amazon AppStream 2.0 documentation
