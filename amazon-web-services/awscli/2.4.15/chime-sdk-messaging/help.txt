NAME
       chime-sdk-messaging -

DESCRIPTION
       The  Amazon  Chime  SDK  Messaging  APIs in this section allow software
       developers to send and receive messages in  custom  messaging  applica-
       tions. These APIs depend on the frameworks provided by the Amazon Chime
       SDK Identity APIs. For more information about the messaging  APIs,  see
       Amazon Chime SDK messaging
