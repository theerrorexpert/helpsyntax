NAME
       push-container-image -

DESCRIPTION
       Push container image for use in a service deployment.

       See 'aws help' for descriptions of global parameters.

SYNOPSIS
            push-container-image
          --service-name <value>
          --image <value>
          --label <value>

OPTIONS
       --service-name  (string) The name of the container service to which the
       container image will be pushed.

       --image (string) The name of the container image to push from the local
       machine to the container service.

       --label  (string)  The label to apply to the registered container image
       on the container service.

       See 'aws help' for descriptions of global parameters.
