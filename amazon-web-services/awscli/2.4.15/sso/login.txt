NAME
       login -

DESCRIPTION
       Retrieves  and  caches an AWS SSO access token to exchange for AWS cre-
       dentials. To login, the requested profile must have  first  been  setup
       using  aws  configure sso. Each time the login command is called, a new
       SSO access token will be retrieved.

       See 'aws help' for descriptions of global parameters.

SYNOPSIS
          login

OPTIONS
       None

       See 'aws help' for descriptions of global parameters.
