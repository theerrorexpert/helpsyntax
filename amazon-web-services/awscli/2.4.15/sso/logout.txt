NAME
       logout -

DESCRIPTION
       Removes  all  cached AWS SSO access tokens and any cached temporary AWS
       credentials retrieved with SSO access tokens across  all  profiles.  To
       use these profiles again, run: aws sso login

       See 'aws help' for descriptions of global parameters.

SYNOPSIS
          logout

OPTIONS
       None

       See 'aws help' for descriptions of global parameters.
