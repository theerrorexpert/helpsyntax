NAME
       autoscaling-plans -

DESCRIPTION
       Use  AWS  Auto Scaling to create scaling plans for your applications to
       automatically scale your scalable AWS resources.
          API Summary

       You can use the AWS Auto Scaling service API to accomplish the  follow-
       ing tasks:

       o Create and manage scaling plans

       o Define  target  tracking  scaling  policies to dynamically scale your
         resources based on utilization

       o Scale Amazon EC2 Auto Scaling groups  using  predictive  scaling  and
         dynamic scaling to scale your Amazon EC2 capacity faster

       o Set minimum and maximum capacity limits

       o Retrieve information on existing scaling plans

       o Access  current  forecast data and historical forecast data for up to
         56 days previous

       To learn more about  AWS  Auto  Scaling,  including  information  about
       granting  IAM  users required permissions for AWS Auto Scaling actions,
       see the AWS Auto Scaling User Guide .

AVAILABLE COMMANDS
       o create-scaling-plan

       o delete-scaling-plan

       o describe-scaling-plan-resources

       o describe-scaling-plans

       o get-scaling-plan-resource-forecast-data

       o help

       o update-scaling-plan
